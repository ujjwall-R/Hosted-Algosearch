
You have been given a binary string s. You have to find the integer number representing the binary string. If the number exceeds INT range then print overflow else print the integer number.
Note the range for INT is from −2,147,483,648 to 2,147,483,647.
Input:

First line will contain T, number of testcases. Then the testcases follow. 
Each testcase contains of a single line of input, a string S of 0s and 1s. 

Output:
For each testcase, output in a single line representing the integer number or the string 'overflow'.
Constraints

1≤T≤1000
1≤len(S)≤104

Sample Input:
2
1001
1011

Sample Output:
9
11
