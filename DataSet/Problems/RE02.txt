
Input description
First line contains an integer T, representing the number of test cases.
For each test case,next line contains an integer X.

Output
For each Test Case, the output contains sequence of 10 integers in format shown in example.
Constraints
T <= 100  
1 <= X <= 10^4 
Example
Input:
1
1439

Output:
1439	4318	2159	6478	3239	9718	4859	14578	7289	21868
