
 
Chef  like  palindrome strings more than normal trings ,so chef Just want to check if characters of a given string can be rearranged to form a palindrome string or not.
 Help him to check, print YES if its possible,else print NO
 
Input
aaaabbbwwww
Output
 YES 
Explanation 
 Possible arrangement is waawbbbwaaw,which is palindrome string.

Constraints
1<=|s<=10^6|,string s contains only lowercase alphabets
