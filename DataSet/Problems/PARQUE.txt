

Parhi's Girlfriend loves words which contains vowels only. They play a game where Parhi would get a string from her and he needs to find out the length of the longest sub-string containing vowels only. If he fails, then his girlfriend would friend-zone him. Help Parhi to impress his girlfriend and not get friend-zoned!
.

Input:
First line contains an integer T denoting the number of test cases.
Next T lines contain a string S. 
Output:
Print an integer denoting the length of the longest substring, that is substring consists of only vowels.
Constraints:

1 ≤ T ≤ 103
1 ≤ |S| ≤ 104

Example:
Input:
1
abcaac
Output: 
2
Explanation:
Longest Substring of S is aa so the answer is 2.
