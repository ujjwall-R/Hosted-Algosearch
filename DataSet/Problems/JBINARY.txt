
Jon is given a decimal number N.
Jon likes a decimal number only if it's binary representation ends with 0.
Help Jon to find out if he likes the given number or not.
Input Format

First line will contain T, number of testcases. Then the testcases follow.
Each testcase contains of a single line of input, one integer N.

Output Format
For each testcase, print a single line containing the integer 1 if Jon likes the given decimal number N or 0 otherwise.
Constraints

1≤T≤106
1≤N≤109


                        Sample Input 1
                        





2
13
14



                        Sample Output 1
                        





0
1


Explanation
13 : 1101. Last bit of 13 is 1. So, Jon doesn't like the number.
14 : 1110. Last bit of 14 is 0. So, Jon likes the number.
