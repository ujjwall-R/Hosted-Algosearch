Given Q queries in each query two numbers N and T is given, You have to  print the value of  sum . Since sum can be large, output it modulo 1000000007 (109+7)  .

Input
The first line of input contains an integer Q denoting the number of queries. 

The next Q lines contains two space separated integers,N and T denoting the parameters of the summation above.


Output

For each query print the value of sum in  a new line . 
  

 
Constraints

1 ≤ Q ≤ 100000
1 ≤ N ≤ 10000000000
1 ≤ T ≤ 1000

T<=100
