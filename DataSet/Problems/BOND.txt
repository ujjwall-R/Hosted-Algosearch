
Problem Name

BOND

Problem Defination :
James has a graph [ Described Below ]



Given a number X , take each digit in turn from left to right . For each digit follow that many black arrows in a row and then follow one white arrow , initial position being node A . James is interested on which node he lands after performing the given operations on X.
Constraints :
1 <= T <= 10^6 1 <= number of digits in X <= 17
Input :

First line contains T, the number of testcases.
The first line of each testcase contains an integer X

Output :

For each testcase output the node at which james is present.

Example :
Input :
191
Output:
A
Explanation

For the given testcase, we start from node A, follow 9 black arrows path : A - B - D - C - E - G - F - A - B - D then a white arrow, which lands james to node F. then 1 black arrow, which lands james to node A. then 1 white arrow, which lands james to same node i.e A.
