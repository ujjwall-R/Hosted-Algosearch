
Jon is single and lonely. He wants to talk to a girl to break his loneliness. 
So, he went to a online chat site and started searching for girls. 
He realised that users with UsernameUsername ended with FF are female users and users with UsernameUsername ended with M are male users. It is guaranteed that an Username will always end with F or M.
Write a program to help Jon find out more female users.
Input Format

First line will contain T, number of testcases. Then the testcases follow.
Each testcase contains of a single line of input, one string Username.

Output Format
For each testcase, output in a single line answer "Yes" if a given Username is of a female user and "No" if a given Username is of a male user.
Constraints

1≤T≤1000
2≤length(Username)≤106


                        Sample Input 1
                        





2
snehaF
rajM



                        Sample Output 1
                        





Yes
No
