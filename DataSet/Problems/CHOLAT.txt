
As we know, Chef has a party and he has many chocolates in which some chocolates are of the same type. He wants to give the chocolates to his friends as perfect gift. i.e. if each friend gets at-least one unique chocolate.
Help Chef to find the maximum number of perfect gifts. 
Input:

First-line will contain TT, the number of test cases. Then the test cases follow: 
Each test case contains two lines of input, the First line contains NN size of the array. 
Second line contains  Arr array of N elements as id of chocolates. 

Output:
For each test case, output in a single line maximum number of perfect gifts.
Constraints

1≤T≤10
1≤N≤104
0≤Arr[i]≤109

Sample Input:
1
 3
 1 2 2   
Sample Output:
2
