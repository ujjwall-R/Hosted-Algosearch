
 
Chef is very interested in image processing. In image processing colours are generally written in hexadecimal.In his project chef is given a decimal number and he has to convert it in hexadecimal.As chef is very lazy he has given this task to you. 
 
Input
The first line contains number of test cases T.
Each line of test case contain a number N
Output
For each test case, output is the hexadecimal equivalent of that number in small letters
 
Constraints

1 ≤ T ≤ 10
1 ≤ N≤  103 

 
Example
Input:
2
255
100
Output:
ff
64
