
Raj is having a pizza party and has ordered some pizzas, each pizza is in for of lower case Latin letters.
We want to find out whether some pizza has the same quantity as the sum of quantities of all other pizzas.
Since he is busy with other preparations, he needs your help to find it.
Input:

The first line of the input contains an integer TT denoting the number of test cases. 
Each of the next T lines contains one string SS consisting of lowercase Latin letters.

Output:
For each test case, output a single line containing "YES" if the condition given above is valid or "NO" otherwise.
Constraints

1≤T≤10001≤T≤1000
1≤len(S)≤1000

Subtasks

Subtask #1 (100 points): Original Constraints

Sample Input:
1

aabcad
Sample Output:
YES

Explantaion:
Pizza named a has count 3 which is equal to count of remaining pizzas. Therefore answer is YES.
