
 
You are given a number N.
  You have to print the square of that number on a new line

 
Input
Input description.

First line will contain a single integer T denoting
the number of integers.Next T lines will contain
a single integer N.
 
Output

Output a single integer for each N, the square of the
  number on a new line.


 
Constraints

1 ≤ T ≤ 100000
1 ≤ N ≤ 100000

 
Example
Input:
3
5
7
2

Output:
25
49
4

 
Explanation
Example case 1.
Simply print the square of the given number on a new line.
