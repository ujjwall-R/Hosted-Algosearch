
 Poppy is a troll and lives in troll town and she is princess of troll town. Trolls have a property that they are always happy untill this problem comes:
Princess Poppy has N chocolates and she does not like chocolates much, but she don't want to throw those chocolates. So she decides to distribute them among her friends but she also wants that everyone gets equal number of chocolates.
If everybody does not get equal number of chocolates she became unhappy.

Poppy is trying to make herself happy and also don't want to waste any chocolate. So she distributes the chocolate and then eat any number of chocolates left.
Help her so that she is happy and eats minimum number of chocolates(as she don't like them).

 
Input
First line of the input contains an integer T, denoting the number of inputs.
T lines follow each containing 2 integers N and M are respectively number of chocolates and number of friends poppy have.

 
Output 

For each test case output the p how many chocolates princess poppy has to eat for making herself happy in a newline.


 
Constraints and Subtasks
Should contain all the constraints on the input data that you may have. Format it like:

Subtask 1: 30 points
1 ≤ T ≤ 20
1 ≤ M < N ≤ 20000

Subtask 2: 70 points
1 ≤ T ≤ 1000
1 ≤ M < N ≤ 2000000


 
Example
Input:
2
7 3
678 34

Output:
1
32
