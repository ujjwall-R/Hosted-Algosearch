
Input description
First line contains an integer T, representing the number of test cases.For each test case,next line contains an integer X.
Output
For each Test Case, the output contains one integer.
Constraints
T <= 100 
1 <= X <= 10^4

Example
Input:
2
58
100

Output:
2860792
25164150
