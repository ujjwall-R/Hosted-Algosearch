
 A child needs help to learn counting. As a professional software engineer, it should be your responsibility to help others through your software. You need to take input in digits and print the equivalent word representation for the number in American name.

 
Input
 First line denotes 'T' number of test cases
It will be followed by 'T' lines containing a whole number
Output
 Print 'T' lines containing equivalent word representation of the specified number in American word with all in lowercase with delimiters as space
Constraints

1 ≤ T ≤  100 
1 ≤ n ≤  10^18 

 
Example
Input:
1
555

Output:
five hundred fifty five

 
Explanation
Testcase 1The output is representation of 555 in American name.
