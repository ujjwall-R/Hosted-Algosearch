

Monica has many toys and she has a weird obsession of arranging all the toys in a line such that their heights are in non-decreasing order. If the order is messed then she gets angry. You know the number of toys and their current arrangement, can you tell if she is angry or not?

Input
First line will contain n the number of toys Monica has.
Next line contains n integers, denoting the heights of the toys in line.

Output

If Monica is angry print "Angry" (without quotes) else print "Happy".

Constraints

1 ≤ n ≤ 105
1 ≤ Ai ≤ 106

Example
Input 1:
5
1 3 5 5 10

Output 1:
Happy

Input 2:
5
1 5 12 7 10

Output 2:
Angry

 
Explanation
Example case 1. 
All the toys are arranged in non-decreasing order hence Monica is "Happy".
Example case 2. 
The toys are not arranged in non-decreasing order hence Monica is "Angry".
