

Rahul loves number so much that whenever his teacher ask him any question he replies with a number. If answer given by rahul is even which means YES if his answer is odd which means NO. His teacher got irritated by him due to his weird ans. So he ask you to translate all the number into YES and NO.
Input

First line contains number of test cases T. T lines follow.
Each test case contains an integer N.

Output

For each test case print "YES" or "NO" without quote.

Constraints

1 ≤ T ≤ 1000
1 ≤ N ≤ 109

Example input
3
4
5
6

Example output
YES
NO
YES
