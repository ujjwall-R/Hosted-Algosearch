
 
Mr.DBShah is the creater of this beautiful place. So being the creater of these island Mr.DBShah has the power take travels from x to 2x island in return of 2 pound and Mr.Rudani who is best friend of Mr.DBShah and help him to create this beautiful land so he has power to travel x to x+1 island in return of 1 pound.
A group of n island (1 to 10^3) created by Mr.DBShah and Mr.Rudani, each one more beautiful and mystical than other one ,widely distributed in the kernighan ocean.An adventure Voyager, Mr.Thakkar come this place and after know about this place he want to visit Q(1 to 10^3) island and you are the guide of Mr.Thakkar and help him to visit these island which they want to visit in minimum pound.
Input
1st line contain number of test cases (t<=1000).
then t line each line contain number of Island(n<=1000)
 
Output
Print minimum cost in Pounds for visit Q island.
 
 
Example
Input:
2
11
13

Output:
8
8
