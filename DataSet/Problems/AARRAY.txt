
Alice has a non-negative integer array of size N. She asks Bob to find the start and end index(1-based) of smallest continuous subarray such that if she sorts that subarray then the complete array will be sorted. Print -1 if no such subarray exist.
Input:

First line will contain N, number of testcases. 
Second line contains N space separated non-negative integers. 

Output:
For each testcase, output space separated start and end of continuous subarray
Constraints

1≤N≤106
0≤arr[i]≤109

Subtasks
Subtask #1(100 points): Original Constraints
Sample Input:
5
1 2 5 3 7

Sample Output:
3 4

EXPLANATION
If the subarray [5,3] is sorted then complete array is sorted.
