
Input description
First line contains an integer N, Number of Test cases.
For each test case,next line will contain a string S(consisting of two space separated words)
Output
For each Test Case, the output contains a boolean answer which may or may not be followed by a integer(space separated).
Constraints
1 <= N <= 100 
1 <= |W| <= 103 
where |W| is the length of each word.
Example
Input:
2
Goonj UIET
axucz uxcaz

Output:
False
True 3


Explanation
When input string is Goonj UIET output is False and inputting axucz uxcaz results in True 3.
