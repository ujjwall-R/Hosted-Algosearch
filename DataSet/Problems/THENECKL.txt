
Chef wants to buy a necklace for Chefina. The necklace will consist of N beads joined together in a circular manner, i.e., the 1st bead will lie between Nth and 2nd beads. The 2nd bead will lie between 3rd and 1st beads and so on. Each bead has a certain colour (some beads can have the same colour). Chef considers the necklace good if no two neighbouring beads have the same colour. The cost of the necklace is equal to the number of distinct colours present in the necklace. 
You have to tell Chef what's the minimum he needs to spend to buy a good necklace. 
Input:

First line will contain T, the number of test cases. Then T the test cases follow. 
Each test case contains a single integer N.

Output:
For each test case, output in a single line the required answer.
Constraints

1≤T≤106
1≤N≤1018

Sample Input:
3
2
3
4  
Sample Output:
2
3
2
