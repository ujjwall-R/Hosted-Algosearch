
Given a string ss consisting of only lowercase characters. You can perform following operation exactly once:  

Choose any character from the English alphabets and remove some (or all) occurrences of this alphabet from the given string. 

What is the smallest length of the string that you can obtain.
Input:

First line will contain TT, number of testcases. Then the testcases follow. 
Each testcase contains a string. 

Output:
For each testcase, output minimum length of the string that you can obtain after applying the above operation exactly once.
Constraints

1≤T≤101≤T≤10
1≤|s|≤501≤|s|≤50

Sample Input:
3
abcc
abc
oiuyt  
Sample Output:
2
2
4
