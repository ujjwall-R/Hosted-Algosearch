
saraguru50 loves coding more than anything else. He loves problem solving. He spends more time with his laptop than with his girl friend. saraguru50 will be happy only if he practices coding for at least K consecutive days. Given the coding schedule of saraguru50 for N days. Each day is represented  either as 1 ( meaning he practiced coding on that day ) or 0 ( meaning he went on outing with his girl friend on that day ). Predict whether saraguru50 is happy or not. 
Input

The first line of the input consist of a two integers N and K. 
The second line consist of N numbers, a1, a2, ... an.

Output

Print "Yes" without quotes if saraguru50 is happy, else print "No" without quotes. 

Constraints

1 ≤ N ≤ 1000
0 ≤ K ≤ N
0 ≤ ai ≤ 1

Example
Input:
7 3
1 1 0 1 1 1 1
Output:
Yes
